Design a modern SaaS-style dashboard UI for a Client & Financial Policy Management Platform used by Financial Advisors, Insurance Brokers, Wealth Managers, and Financial Consultants to manage their clients, policies, claims, tasks, and reports.

The interface should feel like a premium fintech CRM platform, similar to modern financial dashboards. The design must be professional, clean, highly organized, and data-driven, with strong emphasis on KPI metrics, charts, and client management tools.

Layout Structure

Create a responsive layout with three main components:

1. Left Sidebar Navigation (collapsible)
2. Top Header Bar
3. Main Content Area

The sidebar should contain icons and navigation labels for:

Dashboard

Client Management

Policy Management

Claims Tracker

Tasks / Follow-ups

Reports & Analytics

Notifications / Alerts

Settings

Include profile avatar and logout option in the header.
